# 🌱 Agro Farmy — Smart Agriculture & Urban Ecosystems

> Growing the Future of Smart Agriculture. Technology and Farming, Perfectly Connected.

Agro Farmy is a high-end sustainable agritech brand bridging modern technology with nature. We empower both urban gardens and industrial-scale farms by providing smart monitoring solutions, premium organic vermicompost, and innovative urban gardening setups.

---

## 🌾 Tech Stack & Architecture

The Agro Farmy launch pad is built on a high-performance, GPU-optimized frontend stack:

- **Core**: React 18 + Vite (fast HMR, lightweight bundling)
- **Styling**: Tailwind CSS (sleek utility-first design system)
- **Motion**: Framer Motion (GPU-accelerated mount staggers, physics-based springs, and digit reels)
- **Interactions**: GSAP (high-fidelity mouse parallax and elastic magnetic button pulls)
- **Performance**: HTML5 Canvas rendering for atmospheric pollen/light particle physics

## 🌿 Core Business Segments

### 1. High-Performance Vermicompost
- **100% Organic & Eco-Friendly**: Rich in earthworms, essential macro/micronutrients, and beneficial soil microbes.
- **Enhanced Growth Dynamics**: Accelerates root system development and increases natural crop yields.
- **High Moisture Retention**: Dramatically reduces irrigation frequency, promoting water conservation.

### 2. Smart Balcony & Container Gardening
- **Urban Micro-Farming**: Turn limited apartment space (balconies, patios) into thriving food sources.
- **Interactive Guides**: Guidance on matching container sizes with crop soil requirements.

### 3. Industrial Rooftop & Terrace Farms
- **Heat Island Mitigation**: Reduces structural building temperatures while growing organic produce.
- **Structural Load Advice**: Lightweight custom soil mixes designed for rooftop setups.

## 🚀 Local Development Setup

To run the project locally:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```
